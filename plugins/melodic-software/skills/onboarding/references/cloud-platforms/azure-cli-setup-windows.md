# Azure CLI Setup

[https://learn.microsoft.com/en-us/cli/azure/install-azure-cli-windows](https://learn.microsoft.com/en-us/cli/azure/install-azure-cli-windows)

```powershell
winget install --id Microsoft.AzureCLI -e --source winget
```

After installing, you'll want to run `az login` to login with your Azure account.
