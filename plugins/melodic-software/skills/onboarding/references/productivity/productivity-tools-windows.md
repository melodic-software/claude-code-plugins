# Productivity Tools

## Snagit

[https://www.techsmith.com/download/snagit/](https://www.techsmith.com/download/snagit/)

```powershell
winget install --id TechSmith.Snagit.2025 -e --source winget
```

## Razer Synapse 4

**Only install if you have Razer peripherals** (keyboard, mouse, headphones).

[https://www.razer.com/synapse-4](https://www.razer.com/synapse-4)

Razer Synapse enables:

- Custom key bindings and macros
- RGB lighting control
- Device-specific settings and profiles
- Performance tuning for gaming peripherals

```powershell
winget install --id RazerInc.RazerInstaller.Synapse4 -e --source winget
```
