# Figma Setup Windows

Figma is a collaborative design and prototyping tool used for UI/UX design, wireframing, and design handoff between designers and developers.

## Figma Desktop App

Install the Figma desktop application:

[https://www.figma.com/downloads/](https://www.figma.com/downloads/)

```powershell
winget install --id Figma.Figma -e --source winget
```

## Figma Agent

Install the Figma Agent (enables font access and other local integrations). This may not be needed.

```powershell
winget install --id Figma.FigmaAgent -e --source winget
```
