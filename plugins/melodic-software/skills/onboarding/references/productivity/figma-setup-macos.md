# Figma Setup macOS

Figma is a collaborative design and prototyping tool used for UI/UX design, wireframing, and design handoff between designers and developers.

## Install via Homebrew

```bash
brew install --cask figma
```

## Alternative: Direct Download

If you prefer not to use Homebrew, download the macOS installer directly from:

[https://www.figma.com/downloads/](https://www.figma.com/downloads/)
