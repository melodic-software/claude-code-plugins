# Security Tools

You should use the password vault / secrets management system, and VPN solution specified by your manager.

## Keeper Desktop

[https://www.keepersecurity.com/download.html?t=d](https://www.keepersecurity.com/download.html?t=d)

```powershell
winget install --id KeeperSecurityInc.KeeperPasswordManager -e --source winget
```

---

## NordVPN

This is an optional install. Normally companies provide their own VPN solution - but this may be an option for your personal dev machine. There are plenty of VPN alternatives.

[https://nordvpn.com/download](https://nordvpn.com/download)

```powershell
winget install --id NordVPN.NordVPN -e --source winget
```
