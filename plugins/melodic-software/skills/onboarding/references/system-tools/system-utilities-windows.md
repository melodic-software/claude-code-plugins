# System Utilities

## 7-Zip

[https://7-zip.org/download.html](https://7-zip.org/download.html)

```powershell
winget install --id 7zip.7zip -e --source winget
```

---

## Notepad++

[https://notepad-plus-plus.org/downloads/](https://notepad-plus-plus.org/downloads/)

```powershell
winget install --id Notepad++.Notepad++ -e --source winget
```

You'll probably want to open up the app and use the built in plugin manager to install any XML or JSON related plugins (formatting, viewing, etc.)
