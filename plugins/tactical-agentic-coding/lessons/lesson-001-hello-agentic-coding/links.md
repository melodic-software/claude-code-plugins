- [Vite Documentation](https://vitejs.dev/) - Official documentation for Vite - next generation frontend tooling with fast HMR and optimized build

- [pytest Documentation](https://docs.pytest.org/en/stable/) - Official documentation for pytest - the Python testing framework for writing better programs

- [npm Documentation](https://docs.npmjs.com/) - Official documentation for npm - the Node.js package manager and world's largest software registry

- [Bun Documentation](https://bun.sh/docs) - Official documentation for Bun - all-in-one JavaScript runtime with built-in bundler, transpiler, and package manager

- [GitHub CLI Documentation](https://cli.github.com/) - Official documentation for GitHub CLI (gh) - bring GitHub to your terminal for seamless workflow
